# LoanTap Credit Risk Analysis

## Project Overview

This project analyzes customer creditworthiness for LoanTap using machine learning and logistic regression techniques.

The objective is to build an underwriting model capable of predicting whether a borrower is likely to fully repay a loan or default, helping LoanTap improve lending decisions and reduce non-performing assets (NPAs).

---

# Business Problem

LoanTap wants to optimize its loan underwriting process by identifying high-risk applicants before loan disbursement.

The analysis focuses on:

- Predicting loan repayment behavior
- Identifying high-risk customers
- Reducing loan defaults
- Improving lending decisions
- Balancing precision and recall tradeoffs

The project simulates a real-world financial risk analytics workflow used in banking and fintech industries.
